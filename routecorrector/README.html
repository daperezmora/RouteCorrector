Overview

RouteCorrector is a QGIS plugin designed to correct GPS drift and spatial dispersion in mobile environmental monitoring data. It allows users to realign scattered measurement points (e.g., air quality, noise, or particulate matter) along user-defined reference routes, ensuring that mobile sensor data accurately follow real-world paths.

Main Features

  • Define start and end anchor points for correction.
  • Apply two correction modes:
    • Uniform Motion (MRU): preserves temporal progression.
    • Distance-based: preserves original spatial distances.
  • Interactive interface fully integrated in QGIS.
  • Export corrected data for further analysis.

Usage Summary

  1. Load a point layer with your GPS-based measurements.
  2. Create or load a reference route (line layer).
  3. Select the start and end anchor points.
  4. Choose a correction mode: Uniform Motion or Distance-based.
  5. Click Correct Section to realign the data points.
  6. Optionally export the corrected layer.
